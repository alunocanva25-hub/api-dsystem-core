# DSYSTEM SERVER CORE — pacote para GitHub

Versão base: `DSYSTEM_SERVER_CORE_V1.0.1.3_COMPAT_LOGIN_GO_TOKEN`.

Este pacote foi filtrado para subir no GitHub. Não inclui banco local, `.env`, ambiente virtual, cache, logs ou backups.

## Depois de clonar

1. Copie `.env.example` para `.env` localmente.
2. Ajuste as variáveis se necessário.
3. Rode `RUN_LOCAL.bat` para teste local ou `INICIAR_SERVIDOR_REDE.bat` para modo rede.

## Não subir depois

- `.env`
- arquivos `.db`, `.sqlite`, `.sqlite3`
- `.venv/`, `venv/`
- `logs/`, `backups/`
